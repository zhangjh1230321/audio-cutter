## 本地使用

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

### 3. 打开浏览器

访问 **http://localhost:8765**

---

## 操作步骤

### 第一步：上传文件
- 拖拽音频/视频文件到上传区域，或点击选择文件
- 支持格式：MP3、WAV、FLAC、MP4、MOV、MKV 等

### 第二步：调整参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| 静音阈值 | -35 dB | 越小越严格，检测到的气口越少 |
| 最小气口时长 | 0.10 秒 | 低于此时长的静音不算气口 |
| 合并间隔 | 0.30 秒 | 两个气口距离小于此值会合并 |
| 边界保留 | 0.08 秒 | 剪切时两端保留的缓冲 |

### 第三步：分析
- 点击 **「🔍 开始智能分析」**
- 等待分析完成，查看波形图和气口列表

### 第四步：预览调整
- **点击气口列表的「✘ 剪除 / ✔ 保留」** 可切换单个气口
- **点击时间轴上的红色区域** 也可切换
- 点击 **▶ 播放** 可实时预览（自动跳过气口）

### 第五步：导出
- 点击 **「📦 导出剪辑视频」**
- 等待导出完成后点击下载
